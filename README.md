# Magelearn_MinicartOpen
Auto Open minicart after adding products to cart Magento 2

## Features

#### 1. Add custom JS in PHTML Magento2.

#### 2. Modify minicart JS.

#### 3. Check requirejs-config.js for shim configuration.
